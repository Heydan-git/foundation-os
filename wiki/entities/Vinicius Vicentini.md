# Vinicius Vicentini

## What

Developpeur bresilien, createur de [[vinihvc-shark-ui|Shark UI]] — bibliotheque open-source multi-framework (React / Vue / Svelte / Solid) de 110+ composants UI accessibles.

## Why It Matters

Shark UI positionne une **alternative shadcn/ui multi-framework** via [[Ark UI]] (primitives headless non-Radix). Interessant comme signal ecosystem : demontre la demande pour des shadcn-like portes vers Vue/Svelte/Solid. Pour Foundation OS (React only), positioning hors scope direct mais source d'inspiration code MIT.
