bash scripts/wiki-health.sh 2>&1

=== ACTIONS (si problemes) ===

1. HOT.MD STALE (> 3 jours) :
- Lis wiki/log.md + git log --oneline -10
- Reecris wiki/hot.md au format : Last Updated / Key Recent Facts / Recent Changes / Active Threads / Next Action
- Max 500 mots

2. WIKILINKS CASSES :
Pour chaque wikilink casse :
a) Extrais le basename du lien
b) Cherche : find wiki/ docs/ -name "BASENAME*"
c) Si fichier trouve avec nom similaire → corrige le wikilink
d) Si INTROUVABLE → retire le wikilink du contenu (pas du frontmatter)
e) Note dans wiki/meta/lessons-learned.md

3. INDEX STATS DESYNC :
- Count reel : find wiki/ -name "*.md" -not -path "*/meta/templates/*" | wc -l
- Mettre a jour "Total pages: N" dans wiki/index-wiki.md

4. CARTE ORPHELINS :
- Verifier chaque [[lien]] dans foundation-os-map.md → fichier existe ?
- Si absent → retirer. Si nouveau fichier wiki/ pas dans carte → ajouter.

5. FRONTMATTER AUDIT :
- Verifier YAML parseable sur chaque page wiki/concepts/ entities/ sources/
- Si invalide → noter dans thinking.md (NE PAS modifier le frontmatter)

=== SCOPE ===
Modifie UNIQUEMENT wiki/. NE TOUCHE PAS CLAUDE.md, CONTEXT.md, docs/, modules/, scripts/

=== COMMIT ===
git add wiki/
