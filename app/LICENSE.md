## Open Core Model

Foundation OS follows an open core model:

- **Open Source Core** (MIT License): Engine, basic components, MCP orchestration
- **Commercial Features** (Separate License): Enterprise integrations, advanced security, priority support

This model ensures sustainable development while keeping the core freely available to the community.
