# .raw/articles/

Sources brutes articles / blogs / newsletters avant ingestion wiki.

## Convention

- Nommer `YYYY-MM-DD-<slug>.md` ou `<slug>.pdf`
- Drop ici → `wiki-ingest .raw/articles/<file>` ou skill `claude-obsidian:wiki-ingest`
- Apres ingestion, le fichier reste ici (immutable). La page produite vit dans `wiki/sources/` ou `wiki/domains/<domain>/sources/`.
