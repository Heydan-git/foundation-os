# .raw/images/

Screenshots, diagrammes, images a attacher aux pages wiki.

## Contenu attendu

- Screenshots apps (Figma, terminal, browser)
- Diagrammes d'architecture
- Captures graphes (trading charts, dashboards)
- Photos notes manuscrites (scan rapide avant OCR)

## Convention

- Format prefere : PNG, JPG
- Nommage : `YYYY-MM-DD-<slug>.png` ou `<topic>-<desc>.png`
- Attacher depuis une page wiki via `![[.raw/images/<file>]]` (wikilink Obsidian vers fichier repo)
