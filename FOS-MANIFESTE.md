## 🎯 Vision

**Foundation OS** est un système d'exploitation de travail IA-driven qui transforme la collaboration humain-IA en une synergie productive optimale.

### Mission Statement
*Créer le premier OS de travail où l'intelligence artificielle amplifie la créativité humaine sans la remplacer, où chaque décision est tracée, et où la coopération prime sur l'exploitation.*

---

## 🏗️ Principes Fondamentaux

### 1. Coopération > Exploitation
- L'IA est un partenaire, pas un outil
- Décisions prises ensemble, jamais imposées
- Respect mutuel des forces (créativité humaine + puissance computationnelle IA)

### 2. Traçabilité Totale
- Chaque décision documentée avec contexte
- Historique complet accessible (FOS-JOURNAL.md)
- ADR (Architecture Decision Records) pour toutes les décisions techniques

### 3. MD-First Workflow
- Documentation = source de vérité
- Code = contrôleur d'affichage
- Pattern obligatoire : MD → JSX, jamais l'inverse

### 4. Qualité > Vitesse
- Zéro régression tolérée
- Verification systématique avant livraison
- Patterns de qualité intégrés (hooks, reviews)

### 5. Évolution Continue
- Système qui apprend de ses erreurs
- Amélioration des patterns via feedback loops
- RALPH Loop (Read → Analyze → Learn → Propose → Harmonize)

## 🎨 Design Philosophy : Void Glass

### Aesthetic
- **Background** : #06070C (void black)
- **Accent** : #5EEAD4 (glass teal)
- **Depth** : rgba(255,255,255,.025) cards + blur(80px) orbs
- **Typography** : Figtree (UI) + JetBrains Mono (code/labels)

### Symbolism
- **Void** : Espace infini de possibilités
- **Glass** : Transparence, clarté, fragile beauté
- **Contrast** : Dark + light = harmony of opposites

### Forbidden
❌ #0A0A0B, #08080A (trop proches)
❌ Inter, Outfit, system-ui (trop communs)
❌ Opacité > .1 sur fond (perte d'effet glass)
