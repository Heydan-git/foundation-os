## 🎯 Quick Start

### One-Line Installation
```bash
npx create-foundation-os-app my-workspace
cd my-workspace && npm start
```

### Docker Setup
```bash
docker run -p 3000:3000 foundationos/foundation-os:latest
```

### Manual Setup
```bash
# Clone the repository
git clone https://github.com/foundation-os/foundation-os.git
cd foundation-os

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Edit .env.local with your settings

# Start development
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and experience the future of work.

## 🏗️ Architecture

Foundation OS is built on a modern, scalable architecture:

```
┌─────────────────────────────────────────────────────────┐
│                    Foundation OS                        │
├─────────────────────────────────────────────────────────┤
│  React 18 + TypeScript + Tailwind CSS (Void Glass)    │
├─────────────────────────────────────────────────────────┤
│           MCP Orchestrator (250+ Tools)                │
├─────────────────────────────────────────────────────────┤
│  Notion │ Asana │ Figma │ Chrome │ Computer Use │ ... │
├─────────────────────────────────────────────────────────┤
│              Supabase (PostgreSQL + Real-time)         │
├─────────────────────────────────────────────────────────┤
│         Vercel (Global CDN + Edge Functions)           │
└─────────────────────────────────────────────────────────┘
```

### Core Technologies
- **Frontend**: React 18.3, TypeScript 5.4, Tailwind CSS
- **Backend**: Supabase (PostgreSQL + real-time subscriptions)
- **Build**: Vite 5.2 for lightning-fast development
- **Deployment**: Vercel with global CDN
- **AI Integration**: Claude Sonnet 4 + custom model orchestration
